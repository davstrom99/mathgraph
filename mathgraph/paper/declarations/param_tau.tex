\section{Prior Scale Declaration}
\label{def:tau}

\subsection{Input and Assumption Nodes}
No upstream mathematical node is required. This declaration introduces the positive prior scale node \texttt{param.tau}.

\subsection{Declaration}
The prior scale is
\[
  \tau \in \mathbb{R}_{>0}.
\]

\subsection{Result}
The node \texttt{param.tau} supplies the scale parameter used by the Gaussian prior on \(\beta\).
