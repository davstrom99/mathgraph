\section{Response Vector Declaration}
\label{def:response-vector}

\subsection{Input and Assumption Nodes}
No upstream mathematical node is required. This declaration introduces the observed response node \texttt{var.y}.

\subsection{Declaration}
The response vector is
\[
  y \in \mathbb{R}^{n}.
\]
Entry \(y_i\) is the scalar response paired with covariate row \(x_i^\top\).

\subsection{Result}
The node \texttt{var.y} is the observed quantity whose distribution is modeled.
