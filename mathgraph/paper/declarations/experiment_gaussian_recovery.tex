\section{Gaussian Recovery Experiment}
\label{experiment:gaussian-recovery}

\subsection{Input and Assumption Nodes}
Input node: \texttt{validation.synthetic\_recovery}. The experiment uses fixed sample sizes and random seeds.

\subsection{Declaration}
The experiment runs synthetic recovery for
\[
  n\in\{20,50,100,250,500\}.
\]

\subsection{Result}
The node \texttt{experiment.gaussian\_recovery} generates a JSON summary and PNG error plot.
