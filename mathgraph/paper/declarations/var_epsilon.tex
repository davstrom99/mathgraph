\section{Noise Vector Declaration}
\label{def:noise-vector}

\subsection{Input and Assumption Nodes}
No upstream mathematical node is required. This declaration introduces the noise vector node \texttt{var.epsilon}.

\subsection{Declaration}
The noise vector is
\[
  \epsilon \in \mathbb{R}^{n}.
\]

\subsection{Result}
The node \texttt{var.epsilon} supplies the random perturbation object used by observation-model assumptions.
