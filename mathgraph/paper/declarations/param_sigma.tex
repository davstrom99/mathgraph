\section{Noise Scale Declaration}
\label{def:sigma}

\subsection{Input and Assumption Nodes}
No upstream mathematical node is required. This declaration introduces the positive noise scale node \texttt{param.sigma}.

\subsection{Declaration}
The noise scale is
\[
  \sigma \in \mathbb{R}_{>0}.
\]

\subsection{Result}
The node \texttt{param.sigma} supplies the scale parameter used by Gaussian and Student-t noise assumptions.
