\section*{Evidence partition and index construction}
Each path receives exactly one precedence-ordered classification, so
\begin{equation}
F=A\,\dot\cup\,G\,\dot\cup\,E.
\label{deriv:init-evidence-partition}
\end{equation}
Semantic records are constructed only for the review mode's active classes; all remaining generated evidence is represented by deterministic member and family hashes.
