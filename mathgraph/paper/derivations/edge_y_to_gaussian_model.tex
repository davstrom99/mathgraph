\section{Response Vector to Gaussian Model}
\label{edge:y-to-gaussian-model}

\subsection{Input and Assumption Nodes}
Input node: \texttt{var.y}. Result node: \texttt{model.gaussian\_observation}.

\subsection{Derivation}
The model assigns a probability distribution to the observed response vector \(y\).

\subsection{Result}
The response vector is the modeled observation in the Gaussian linear model.
