\section{Design Matrix to Student-t Model}
\label{edge:X-to-student-t-model}

\subsection{Input and Assumption Nodes}
Input node: \texttt{var.X}. Result node: \texttt{model.student\_t\_noise}.

\subsection{Derivation}
The Student-t observation model uses each row \(x_i^\top\) of \(X\) to form the linear location \(x_i^\top\beta\).

\subsection{Result}
The design matrix is a required input to the Student-t observation model.
