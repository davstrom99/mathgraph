\section{Response Vector to Student-t Model}
\label{edge:y-to-student-t-model}

\subsection{Input and Assumption Nodes}
Input node: \texttt{var.y}. Result node: \texttt{model.student\_t\_noise}.

\subsection{Derivation}
The Student-t observation model specifies the conditional distribution of the existing response vector:
\[
  y_i \mid X,\beta,\sigma,\nu \sim t_\nu(x_i^\top\beta,\sigma).
\]

\subsection{Result}
The response vector is the observed object whose conditional distribution is specified by the Student-t model.
