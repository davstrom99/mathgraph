\section{MAP from Objective}
\label{deriv:map-from-objective}

\subsection{Input and Assumption Nodes}
Input node: \texttt{objective.gaussian\_negative\_log\_posterior}.

\subsection{Derivation}
The MAP estimator is the minimizer of the negative log posterior objective:
\[
  \hat{\beta}_{MAP} =
  \arg\min_\beta
  \frac{1}{2\sigma^2}\|y-X\beta\|_2^2
  +
  \frac{1}{2\tau^2}\|\beta\|_2^2.
\]

\subsection{Result}
The result node is \texttt{estimator.beta\_map}.
