\section{Beta to Student-t Model}
\label{edge:beta-to-student-t-model}

\subsection{Input and Assumption Nodes}
Input node: \texttt{param.beta}. Result node: \texttt{model.student\_t\_noise}.

\subsection{Derivation}
The Student-t observation model uses \(\beta\) in the linear location:
\[
  y_i = x_i^\top\beta + \epsilon_i.
\]

\subsection{Result}
The coefficient vector is required to specify the Student-t observation model.
