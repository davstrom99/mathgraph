\section{Design Matrix to Gaussian Model}
\label{edge:X-to-gaussian-model}

\subsection{Input and Assumption Nodes}
Input node: \texttt{var.X}. Result node: \texttt{model.gaussian\_observation}.

\subsection{Derivation}
The Gaussian observation model uses each row \(x_i^\top\) of \(X\) to form the linear mean \(x_i^\top\beta\).

\subsection{Result}
The design matrix is a required input to the Gaussian observation model.
