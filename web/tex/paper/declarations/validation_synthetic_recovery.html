Synthetic Recovery Validation

Input and Assumption Nodes

Input node: estimator.beta_map. Data are simulated from model.gaussian_observation.

Declaration

For each sample size \(n\) and seed, the validation computes

\[ \|\hat{\beta}_{MAP,n}-\beta_0\|_2. \]

Result

The node validation.synthetic_recovery checks whether mean estimation error decreases as \(n\) increases.
