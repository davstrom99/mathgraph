Design Matrix Declaration

Input and Assumption Nodes

No upstream mathematical node is required. This declaration introduces the observed design matrix node var.X.

Declaration

The design matrix is

\[ X \in \mathbb{R}^{n \times d}. \]

Row \(x_i^\top\) contains the \(d\)-dimensional covariates for observation \(i\).

Result

The node var.X supplies the covariates used by the observation model.
