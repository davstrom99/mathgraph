MAP Estimator for Beta

Input and Assumption Nodes

Input nodes: objective.gaussian_negative_log_posterior and posterior.gaussian_conjugate.

Declaration

The estimator is

\[ \hat{\beta}_{MAP} = \arg\min_\beta \frac{1}{2\sigma^2}\|y-X\beta\|_2^2 + \frac{1}{2\tau^2}\|\beta\|_2^2. \]

Result

For the Gaussian conjugate model, the MAP estimator equals the posterior mean \(\mu_n\).
