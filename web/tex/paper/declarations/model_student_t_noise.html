Student-t Noise Model

Input and Assumption Nodes

Inputs are var.X, var.y, param.beta, var.epsilon, param.sigma, and param.nu. This node is an alternative observation-model branch over shared variables.

Declaration

The Student-t observation model is

\[ y = X\beta + \epsilon, \qquad \epsilon_i \sim t_\nu(0,\sigma). \]

Result

The node model.student_t_noise represents a heavy-tailed observation-model branch.
