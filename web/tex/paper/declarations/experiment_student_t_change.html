Student-t Change Experiment

Input and Assumption Nodes

Input node: approx.student_t_map. The experiment compares Gaussian and Student-t MAP behavior.

Declaration

The experiment simulates heavy-tailed data from model.student_t_noise, then compares

\[ \|\widehat{\beta}_{G}-\beta_0\|_2 \quad\text{and}\quad \|\widehat{\beta}_{t}-\beta_0\|_2 \]

across fixed random seeds.

Result

The node experiment.student_t_change generates a JSON summary and PNG plot.
