Synthetic Recovery Validation Derivation

Input and Assumption Nodes

Input node: estimator.beta_map. Data are generated from model.gaussian_observation.

Derivation

For each \(n\), seed, and true coefficient \(\beta_0\), compute

\[ e_n = \|\hat{\beta}_{MAP,n}-\beta_0\|_2. \]

Average \(e_n\) across seeds.

Result

The validation result is a decreasing mean error trend as sample size increases.
