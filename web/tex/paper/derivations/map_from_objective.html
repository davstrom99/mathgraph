MAP from Objective

Input and Assumption Nodes

Input node: objective.gaussian_negative_log_posterior.

Derivation

The MAP estimator is the minimizer of the negative log posterior objective:

\[ \hat{\beta}_{MAP} = \arg\min_\beta \frac{1}{2\sigma^2}\|y-X\beta\|_2^2 + \frac{1}{2\tau^2}\|\beta\|_2^2. \]

Result

The result node is estimator.beta_map.
