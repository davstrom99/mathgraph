Noise Vector to Gaussian Model

Input and Assumption Nodes

Input node: var.epsilon. Result node: model.gaussian_observation.

Derivation

The Gaussian observation model uses the noise vector \(\epsilon\) in

\[ y = X\beta + \epsilon, \qquad \epsilon \sim \mathcal{N}(0,\sigma^2 I_n). \]

Result

The noise vector is a required object for the Gaussian observation model.
