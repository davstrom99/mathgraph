Gaussian Negative Log Posterior Derivation

Input and Assumption Nodes

Input node: posterior.gaussian_conjugate.

Derivation

Dropping constants independent of \(\beta\), the negative log posterior is

\[ \frac{1}{2\sigma^2}\|y-X\beta\|_2^2 + \frac{1}{2\tau^2}\|\beta\|_2^2. \]

Result

The result node is objective.gaussian_negative_log_posterior.
