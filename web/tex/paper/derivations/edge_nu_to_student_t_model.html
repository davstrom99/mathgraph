Degrees of Freedom to Student-t Model

Input and Assumption Nodes

Input node: param.nu. Result node: model.student_t_noise.

Derivation

The Student-t observation model uses the positive degrees of freedom \(\nu\) in

\[ \epsilon_i \sim t_\nu(0,\sigma). \]

Result

The parameter \(\nu\) controls the tail heaviness of the Student-t noise distribution.
