Student-t Change Experiment Derivation

Input and Assumption Nodes

Input node: approx.student_t_map.

Derivation

The experiment compares coefficient recovery under Gaussian MAP and Student-t MAP when data are generated with heavy-tailed residuals:

\[ \|\widehat{\beta}_{G}-\beta_0\|_2 \quad\text{versus}\quad \|\widehat{\beta}_{t}-\beta_0\|_2. \]

Result

The experiment will generate results/student_t_change_summary.json and results/student_t_change_error.png.
