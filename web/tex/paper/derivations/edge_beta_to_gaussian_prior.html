Beta to Gaussian Prior

Input and Assumption Nodes

Input node: param.beta. Result node: prior.gaussian_beta.

Derivation

The Gaussian prior is a distributional assumption on the existing coefficient vector:

\[ \beta \sim \mathcal{N}(0,\tau^2 I_d). \]

Result

The coefficient vector \(\beta\) is the random variable governed by the Gaussian prior assumption.
