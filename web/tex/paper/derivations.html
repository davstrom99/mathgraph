Variables and Parameters

The observed design matrix is \(X \in \mathbb{R}^{n \times d}\).

The observed response vector is \(y \in \mathbb{R}^n\).

The unknown regression coefficient vector is \(\beta \in \mathbb{R}^d\).

These definitions connect the symbols \(X\), \(y\), and \(\beta\) to the Gaussian observation model.

Gaussian Model

The Gaussian linear observation model is

\[ y = X\beta + \epsilon, \qquad \epsilon \sim \mathcal{N}(0, \sigma^2 I_n). \]

The implementation simulates each row of \(X\) from a standard normal distribution and then forms \(y=X\beta+\epsilon\) with independent Gaussian noise.

The Gaussian prior on \(\beta\) is

\[ \beta \sim \mathcal{N}(0, \tau^2 I_d). \]

The Gaussian likelihood is

\[ p(y \mid X,\beta,\sigma^2) = (2\pi\sigma^2)^{-n/2} \exp\left(-\frac{1}{2\sigma^2}\|y-X\beta\|_2^2\right). \]

Combining the Gaussian likelihood with the Gaussian prior gives the conjugate posterior

\[ p(\beta \mid X,y) = \mathcal{N}(\mu_n,\Sigma_n), \]

where

\[ \Sigma_n = \left(\frac{1}{\sigma^2}X^\top X + \frac{1}{\tau^2}I_d\right)^{-1}, \qquad \mu_n = \Sigma_n \frac{1}{\sigma^2}X^\top y. \]

The implementation forms the posterior precision matrix

\[ \Lambda_n = \frac{1}{\sigma^2}X^\top X + \frac{1}{\tau^2}I_d \]

and solves \(\Lambda_n\mu_n=\sigma^{-2}X^\top y\).

The negative log posterior is the quadratic objective

\[ \frac{1}{2\sigma^2}\|y-X\beta\|_2^2 + \frac{1}{2\tau^2}\|\beta\|_2^2 \]

up to constants independent of \(\beta\).

The MAP estimator minimizes this objective,

\[ \hat{\beta}_{MAP} = \arg\min_\beta \frac{1}{2\sigma^2}\|y-X\beta\|_2^2 + \frac{1}{2\tau^2}\|\beta\|_2^2. \]

For the Gaussian posterior, the posterior mode equals the posterior mean. Equivalently, the MAP estimator solves the same linear system as \(\mu_n\), so the implemented closed-form MAP estimator returns the posterior mean.

Validation and Experiments

The synthetic recovery validation checks whether estimation error decreases as sample size increases under data generated by the Gaussian model. For each sample size \(n\) and random seed, it computes

\[ \|\hat{\beta}_{MAP,n}-\beta_0\|_2. \]

The validation compares the mean error across seeds at small and large sample sizes.

The Gaussian recovery experiment runs the synthetic recovery validation over several sample sizes and random seeds, returning JSON-serializable mean errors, standard deviations, raw errors, parameters, and seeds.

Student-t Change

The Student-t model changes the noise distribution while preserving the linear mean:

\[ y = X\beta + \epsilon, \qquad \epsilon_i \sim t_\nu(0,\sigma). \]

The Student-t likelihood is the product of Student-t densities over residuals \(y_i - x_i^\top\beta\).

The Student-t MAP approximation keeps the Gaussian prior and numerically minimizes the Student-t negative log posterior.

The Student-t change experiment compares the Gaussian MAP and Student-t MAP estimators under heavy-tailed noise.
