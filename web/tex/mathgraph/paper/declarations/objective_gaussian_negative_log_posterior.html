Gaussian Negative Log Posterior

Input and Assumption Nodes

Input node: posterior.gaussian_conjugate. Constants independent of \(\beta\) are omitted.

Declaration

The objective is

\[ \frac{1}{2\sigma^2}\|y-X\beta\|_2^2 + \frac{1}{2\tau^2}\|\beta\|_2^2. \]

Result

The node objective.gaussian_negative_log_posterior is the scalar objective minimized by the Gaussian MAP estimator.
