Student-t Numerical MAP Approximation

Input and Assumption Nodes

Input nodes: likelihood.student_t and prior.gaussian_beta.

Declaration

The approximation minimizes the Student-t negative log likelihood plus the Gaussian prior penalty:

\[ \widehat{\beta}_{t} = \arg\min_{\beta} \left\{ -\log p_t(y\mid X,\beta,\sigma,\nu) + \frac{1}{2\tau^2}\|\beta\|_2^2 \right\}. \]

This objective is optimized numerically; no closed-form conjugate posterior is claimed.

Result

The node approx.student_t_map is an optimization-based MAP approximation, not a conjugate posterior.
