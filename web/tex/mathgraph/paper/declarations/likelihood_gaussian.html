Gaussian Likelihood

Input and Assumption Nodes

Input node: model.gaussian_observation. The variance \(\sigma^2\) is known.

Declaration

The Gaussian likelihood is

\[ p(y\mid X,\beta,\sigma^2) = (2\pi\sigma^2)^{-n/2} \exp\left(-\frac{1}{2\sigma^2}\|y-X\beta\|_2^2\right). \]

Result

The node likelihood.gaussian evaluates the log likelihood for fixed \(X,y,\beta,\sigma\).
