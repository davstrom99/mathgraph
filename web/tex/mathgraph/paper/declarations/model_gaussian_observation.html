Gaussian Observation Model

Input and Assumption Nodes

Inputs are var.X, var.y, param.beta, var.epsilon, and param.sigma. The model assumes iid Gaussian observation noise.

Declaration

The observation model is

\[ y = X\beta + \epsilon, \qquad \epsilon \sim \mathcal{N}(0,\sigma^2 I_n). \]

The implementation samples rows of \(X\) from a standard normal distribution and then forms \(y=X\beta+\epsilon\).

Result

The node model.gaussian_observation defines the data-generating Gaussian linear model.
